# Security Policy

## Supported Versions

We release security updates for the current major version. Please upgrade to the latest release to receive security fixes.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities in public issues.**

If you believe you've found a security issue in Fix My Shit (fms):

1. **Contact the maintainer** — Use the contact details in [package.json](package.json) or reach out via the repository owner.
2. **Include** — A clear description of the vulnerability, steps to reproduce, and potential impact.
3. **Allow time** — We will acknowledge your report and work on a fix. We may follow up with questions.

We appreciate responsible disclosure. Reporters will be credited when the issue is resolved (unless they prefer to remain anonymous).

We do not have a bug bounty program at this time.
